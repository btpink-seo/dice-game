# For Production
```bash
$ npm install
$ npm run build # angular수정하면 아래 명령어 실행해줘야함
$ npm start
```

# For Development
## 초기화
```bash
$ npm install -g @angular/cli@11.0.4
$ npm install -g nodemon
$ npm install 
```

## 서버기동
```bash
$ ng serve # Angular 개발서버 포트번호 4200
$ nodemon ./app.js # 소켓서버 (nodemon: 서버 수정 사항 자동갱신해줌)
```
